Installing on an Android device

  1. Enable installation from unknown sources. Typlically, you would find this in Settings > Security > Unknown Sources on your device.
  2. Visit http://bit.ly/1x2jfNe to download the .apk file. Your browser may warn that this type of file can harm your device. Ignore this warning.
  3. Click on the file once it has downloaded (normally you can click on the download complete notification in the notifications area). Android may display a Complete Action Using prompt. Select Package Installer and follow the prompts to install the app.

Installation instructions also located http://marshallhampson.com/fortnitta

Building the Project in Android Studio

  1. cd into the directory you want the project cloned
  2. Clone the repository. Using SSH (requires setting up SSH): git clone git@github.com:UCDClassNitta/ECS160Android.git Using HTTPS (does not require setup):git clone https://github.com/UCDClassNitta/ECS160Android.git
  3. Visit the newly created folder ECS160Android e.g. cd ECS160Android
  4. Download and install the latest Java JDK (not JRE).
  5. Download and install Android Studio.
  6. Download the latest Gradle. Extract it to a directory and remember the location of this directory. This is the "installation" directory of Gradle.
  7. Open Android Studio. You'll see the text "Welcome to Android Studio" on the quick start screen.
  8. Click Open an existing Android Studio project.

  9. Navigate to the directory where you checked out ECS160Android. Now click inside that directory and select build.gradle. Make sure you've selected build.gradle before clicking OK.

  10. The dialog now switches to Import Project from Gradle. The radio button Use local gradle distribution should be checked (if it isn't you've probably worked with Android Studio before and it knows where your Gradle is) and it will ask you for a Gradle home. Give it the directory that you extracted Gradle to.

  11. You should see a few Building gradle dialogs. These might take a few minutes.

  13. Click Install missing platform(s) and sync project. It should ask you to install the Android SDK API level 20. When it's done, click Finish on the dialog.

  15. You can now run the Android project. If you have an Android phone, plug it in (using a data cable and not a charge-only cable), and enable Android debugging on your phone (usually in Settings -> Developer options). A lot less hassle compared to the emulator.
