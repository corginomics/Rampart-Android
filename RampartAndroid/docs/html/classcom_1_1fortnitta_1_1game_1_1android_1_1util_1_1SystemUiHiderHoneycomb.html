Detailed Description

An API 11+ implementation of SystemUiHider.

Uses APIs available in Honeycomb and later (specifically View#setSystemUiVisibility(int)) to show and hide the system UI.

If the client requested fullscreen, add flags relevant to hiding the status bar. Note that some of these constants are new as of API 16 (Jelly Bean). It is safe to use them, as they are inlined at compile-time and do nothing on pre-Jelly Bean devices.
